# vue_project

## 简介

本项目为基于 Vue 3 + Element Plus 的前端项目，包含弹幕墙、留言板、实用工具箱等功能，配套 Node.js 后端服务，支持弹幕、留言、IP归属地、随机头像等。

## 主要功能

- 弹幕墙：支持彩色弹幕、弹幕速度调节
- 留言板：支持富文本留言、插入图片、显示IP归属地和头像
- 实用工具箱：字符串、编码、颜色、时间、生活等常用工具
- 响应式设计，兼容手机端

## 前端启动

```bash
npm install
npm run dev
```

默认端口：5173
